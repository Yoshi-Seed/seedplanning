# 受注エコシステム（選択でふわっと展開）

「全体の俯瞰」を残したまま、クリックで段階的にノードを強調表示するUIです。

- End Client（Eli Lilly）をクリック → 一次がふわっと表示
- 一次をクリック → 二次（あれば）がふわっと表示、なければそのままSEEDへ
- 空白クリックで解除（俯瞰に戻る）
- リセット / 全て表示 ボタン付き

## 使い方（ローカル）
- `index.html` を開くだけでOK
- もしブラウザ制限が出る場合は簡易サーバーで：
  - `python -m http.server 8000`
  - `http://localhost:8000`

## GitHub Pages
このフォルダをそのままリポジトリに置いて Pages を有効化すれば公開できます。

## データ編集
`data.js` の `ECOSYSTEM.nodes / edges` を編集してください。
